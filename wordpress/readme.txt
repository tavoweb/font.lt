Enables easy use of the Font.lt icons font set from within WordPress.  Icons can be inserted using either HTML, shortcode, or a built-in TinyMCE plugin.
**Aprašymas**

Font.lt projektas "Font ikonos"

Norėdami įtraukti iconas į savo wordpress puslapį naudokite:

`[icon name=icon-ikonos pavadinimas] trumpinį`



##Usage Instructions

To use any of the Font icons on your WordPress site you have three options:

__HTML__

All code examples on the Font site apply: [https://www.font.lt/ikonos/](https://www.font.lt/ikonos/)

**Examples**

Lrt icon

`<i class="icon-lrt"></i>`

Phone icon

`<i class="icon-phone"></i>`

Chevron left icon

`<i class="icon-chevron-left"></i>`

__Shortcode__

Don't want to worry about HTML tags?  You can use a shortcode in your posts, pages and even widgets to display a Font icon.

The shortcode to use is `[icon name=name-of-icon]`, where name=X is the class of the icon you would like to use.

**Examples**

Pencil icon

`[icon name=icon-pencil]`

Phone icon

`[icon name=icon-phone]`

Chevron left icon

`[icon name=icon-chevron-left]`

__TinyMCE Plugin__

Select the icon you would like to add to your post or page content from the Glyphs drop-down menu in the Visual Editor.

![image](https://font.lt/wpplugin/img/Font-Awesome-TinyMCE-plugin.jpg) 


##Installation

1. Upload Font Icons to the `/wp-content/plugins/` directory.

2. Activate the plugin through the 'Plugins' menu in WordPress.

3. Use one of the three usage methods (HTML, Shortcode, or TinyMCE plugin) within the content of your posts or pages.   

4. Use either the HTML or Shortcode methods inside your text widgets.
